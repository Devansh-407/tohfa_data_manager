# 🎁 Tohfa Data Manager - Easy Guide for Sister

## 🌟 What This Tool Does

This is a **simple website** that helps you manage all the product data for The Tohfa Creations website **without any coding knowledge!**

## 🚀 How to Use (Super Easy!)

### Step 1: Open the Tool
1. Find the `TohfaDataManager` folder in your D: drive
2. Double-click on the `index.html` file
3. It will open in your web browser (Chrome, Firefox, etc.)

### Step 2: Add Products
1. Click on the **"Products"** card
2. Fill in the form:
   - Product Name (e.g., "Vintage Rose Jewelry Box")
   - Description (what the product is)
   - Price (just numbers, like 7499)
   - Category (choose from dropdown)
   - Occasion (choose from dropdown)
   - Images (one image URL per line)
   - Features (one feature per line)
3. Click **"Add Product"**
4. Repeat for all products

### Step 3: Create Custom Menus 🎨🎂💝
1. Click on **"Custom Menu"** card
2. Create special menus like:
   - 🎨 Special occasions (Wedding, Anniversary)
   - 🎂 Seasonal menus (Diwali, Christmas)
   - 💝 Custom content (Personalized gifts)
3. Fill in menu name and description
4. Choose a pretty icon emoji
5. Click **"Create Menu"**

### Step 4: Download Files
1. After adding products, click **"Download products.json"**
2. After creating menus, click **"Download custom-menu.json"**
3. The files will download to your computer

### Step 5: Send to Developer
1. Send the downloaded `.json` files to your brother/developer
2. He will update the website with your data
3. ✅ Done! Your website is updated!
